# G&G Transport Inc. 官方网站

这是一个为 G&G Transport Inc. 创建的专业宣传网站，采用现代化的响应式设计。

## 功能特点

- ✅ 响应式设计，支持桌面、平板和移动设备
- ✅ 中英文双语切换
- ✅ 现代化的UI设计
- ✅ 平滑滚动导航
- ✅ 联系表单
- ✅ 移动端友好的导航菜单
- ✅ 动画效果和交互体验

## 文件结构

```
G&G/
├── index.html      # 主页面
├── styles.css      # 样式文件
├── script.js       # JavaScript交互功能
└── README.md       # 说明文档
```

## 使用方法

1. 直接在浏览器中打开 `index.html` 文件即可查看网站
2. 或者使用本地服务器运行（推荐）：
   ```bash
   # 使用 Python
   python -m http.server 8000
   
   # 使用 Node.js (需要安装 http-server)
   npx http-server
   ```
   然后在浏览器中访问 `http://localhost:8000`

## 主要页面部分

- **首页 (Hero Section)**: 展示公司主要信息和行动号召
- **服务项目**: 展示8项核心服务
- **关于我们**: 公司介绍和统计数据
- **为什么选择我们**: 公司优势特点
- **联系我们**: 联系信息和在线表单

## 自定义内容

您可以根据需要修改以下内容：

1. **公司信息**: 在 `index.html` 中修改联系信息、地址等
2. **服务内容**: 在服务卡片部分更新服务描述
3. **颜色主题**: 在 `styles.css` 的 `:root` 变量中修改颜色
4. **表单提交**: 在 `script.js` 中修改表单提交逻辑，连接到后端API
